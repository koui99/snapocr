# SnapOCR 视觉改版 Brief(给设计模型)

## 你的任务
为一个**已开发完成、功能稳定**的 Windows 10 桌面截图/贴图/OCR 工具「SnapOCR / 截图精灵」做**视觉改版**。
当前 UI「能用但一般」,需要更精致。请产出 **6 个自包含的 HTML 效果图文件** + 每版配套的 **design token 色板**。

**交付物(严格按此)**:
- 6 个独立 HTML 文件,文件名:
  - `redesign_A_light.html` / `redesign_A_dark.html` (风格A:现代浅色精致版)
  - `redesign_B_light.html` / `redesign_B_dark.html` (风格B:macOS 风格)
  - `redesign_C_light.html` / `redesign_C_dark.html` (风格C:科技感强对比)
- 每个 HTML **完全自包含**:CSS 内联在 `<style>`,图标用内联 SVG 或 Unicode,**禁止任何外部 CDN/字体/JS/图片链接**(离线可看)。
- 每个 HTML **底部**附一个「Design Token」区块,用表格列出该版的全部 token 值(见下「Token 契约」),方便开发直接落地到 QSS。
- 托盘图标:在每个 HTML 里画出该风格的**托盘图标方案**(48×48 和 16×16 两种尺寸预览),用 SVG。风格自由发挥,但要能在 Windows 任务栏小尺寸下辨识。

## 落地约束(很重要,决定你怎么设计)
本软件用 **PySide6 (Qt6) + QSS** 渲染,主题由「design token + QSS 占位符替换」驱动。所以:
- 你的配色/圆角/间距最终要能映射成下方「Token 契约」里的键。**每版必须给全套 token 值(浅色版和深色版各一套)**。
- 不要依赖 CSS 渐变动画、backdrop-filter、box-shadow 多层堆叠等 QSS 难还原的效果;阴影、圆角、边框、纯色/简单线性渐变可以。
- 字体用系统中文字体栈:`"Microsoft YaHei","PingFang SC",sans-serif`。

## Token 契约(每版浅/深各给一套值)
```
primary            主色(选中/强调/主按钮)
primary_hover      主色悬停
primary_active_bg  主色浅底(导航选中底、按钮按下底)
bg_main            窗口主背景(略深)
bg_card            卡片/内容区背景
text_main          主文字
text_secondary     次要文字
text_muted         弱化文字/提示
border             分隔线/边框
danger             危险色(关闭悬停、删除)
success            成功色
radius_sm/md/lg    圆角:按钮/控件、卡片、窗口(给px)
```

## 当前基线(风格A「现代浅色精致版」请在此基础上打磨,不要推倒)
- primary #2D7FF9 / hover #1B68DC / active_bg #EEF4FF
- bg_main #F5F6F8 / bg_card #FFFFFF
- text #333/#666/#999 / border #E2E4E8 / danger #EF4444 / success #10B981
- 圆角 sm6 md8 lg12;阴影 0 4px 16px rgba(0,0,0,.08)
- 风格定位:克制、专业、像 Win11 现代应用,不花哨

## 需要画的界面(每个 HTML 都要包含这 5 块,用浅灰/深色桌面背景模拟"浮在桌面上")

### ① 截图选区 + 标注工具栏(核心)
- 一张"桌面截图"上拖出矩形选区;选区外半透明遮罩,选区内清晰;四周 8 个控制点 + 主色边框;左上角尺寸 `1280 × 720`。
- 选区右下角外侧两条横向工具条:
  - 上排标注工具(10个):矩形/椭圆/直线/箭头/画笔/马克笔/马赛克/文字/序号/橡皮擦
  - 下排操作:撤销/重做 | 取色 | 钉图/复制/保存/**识别(OCR,稍醒目)**/取消(×)/确认(✓)
- 选中"矩形"时下方二级属性条:粗细三档 + 颜色色板。
- 鼠标处放大镜+取色小窗(像素网格 + 坐标 + HEX)。

### ② 贴图浮窗 + 右键菜单
- 无标题栏纯图片浮窗,柔投影 + 细边框,角上 `100%` 缩放徽标。
- **重要新特性**:贴图浮窗有「彩色边框」(当前是蓝紫橙渐变描边),请给出你这版的边框设计。
- 右键菜单:复制/保存图片/文字识别 | 缩放(放大/缩小/恢复100%)/不透明度▸/旋转 | 置顶✓/取消贴图/销毁。

### ③ OCR 原地文字面板(新功能,重点!当前设计较弱,重点优化)
这是接在**贴图正下方**的一条面板(不是独立窗口):
- 深色圆角底(浅色主题下也用深底以保证白图上文字清晰),整段可选文本(等宽字体排版还原原图版式)。
- 标题行:左侧「收起」按钮 + 提示「Ctrl+A 全选 · Ctrl+C 复制」。
- 收起态:整个深色框消失,只在图下方留一个小药丸按钮「显示文字」。
- 请优化这块的视觉:深色面板怎么和上方图片衔接更自然、按钮怎么更精致。给出展开态 + 收起态两个样子。

### ④ OCR 结果窗(独立窗口,旋转态贴图识别时用)
- 标题栏「文字识别结果」+ 关闭按钮。
- 左原图缩略 + 右可编辑识别文本框。
- 顶部:语言下拉(中英混合/仅英文/日语)+ 重新识别。
- 底部:复制全部/保存txt/关闭;状态栏 `识别耗时 0.8s · 置信度 96%`。

### ⑤ 设置窗 + 托盘菜单
- 设置窗:左侧竖导航(常规/热键/截屏/贴图/输出/文字识别/关于,高亮"热键")+ 右侧内容;热键页几行「功能名 + 快捷键框」(截屏F1/贴图F3/隐藏显示贴图Shift+F3/文字识别F4);底部 恢复默认/取消/确定。
- 托盘菜单(模拟 Windows 右下角弹出小卡片):截屏/贴图(钉剪贴板的图)/文字识别/最近文件▸ | 设置/开机启动✓ | 帮助/关于 | 退出。

## 三版风格定位
- **A 现代浅色精致版**:在现有 #2D7FF9 蓝基础上提升精致度(更好的间距/层次/微妙阴影/图标统一),保持克制专业。
- **B macOS 风格**:仿 macOS Big Sur/Sonoma——更圆润、半透明质感(用纯色近似)、细腻分隔、SF 风格留白、红黄绿交通灯按钮可点到即止。
- **C 科技感强对比**:深色为主调性、高对比、霓虹/电光强调色(青/紫/蓝),适合"工具感/极客"定位,锐利但不刺眼。

## 排版
每个 HTML 从上到下排列 5 块界面,加中文小标题(① …);文件最底部放该版完整 Token 表格。代码整洁、class 语义化。

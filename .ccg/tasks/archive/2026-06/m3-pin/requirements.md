# M3 贴图(Pin)— 需求规格

> 任务: m3-pin · 策略: full-collaborate · Phase 1 研究与分析产物
> 对照 design/BRIEF.md 界面②、design/mockup.html(L556–579 贴图浮窗 + 右键菜单)、design/PLAN.md 里程碑 M3。

## 目标
把一张图片(来自 M2 截图的「钉图」按钮,或系统剪贴板)以**无边框、置顶、带柔和阴影**的浮窗
钉在桌面上。浮窗可拖动、滚轮缩放,右下角显示缩放比例徽标;右键呼出完整菜单
(复制/保存/OCR/缩放/不透明度/旋转/置顶/取消贴图/销毁)。支持多个贴图共存,Shift+F3 一键显隐全部。
替换 M1/M2 中 pin / pin_clipboard / toggle_pin 的占位逻辑,接上真实贴图流程。

## 范围(本轮 M3)

**IN(本轮交付):**
- **贴图浮窗 PinWindow**:无边框 + 置顶(WindowStaysOnTop)+ Tool(不进任务栏)+ 柔和阴影 + 1px 细边框;
  右下角缩放徽标 `100%`(rgba(0,0,0,0.6) 底白字,还原 mockup)。
- **贴图来源**:
  - M2 截图工具栏「钉图」按钮 → 直接把合成图钉到桌面(sig_result(image,"pin"))。
  - F3 / 托盘「贴图」「从剪贴板贴图」→ 读系统剪贴板图片钉出;剪贴板无图时提示不崩。
- **交互**:
  - 拖动浮窗移动(无边框窗整体拖拽)。
  - 滚轮缩放(以窗口中心为基准,徽标实时更新百分比;缩放范围约 10%–800%)。
  - 键盘:Esc 取消贴图、Shift+Esc 销毁、Ctrl+C 复制、Ctrl+S 保存、+/- 缩放。
- **右键菜单**(还原 mockup 分组与分隔线):
  - 复制(Ctrl+C)、保存图片(Ctrl+S)、文字识别(OCR,**M4 占位**) |
  - 缩放(放大/缩小/恢复100%)、不透明度(100/80/60/40/20% 子菜单)、旋转(向左/向右 90° 子菜单) |
  - 置顶(可勾选,默认开)、取消贴图(Esc)、销毁(Shift+Esc,红色)。
- **多窗管理 PinManager**:统一创建/定位/跟踪所有贴图;关闭即从管理器移除(无泄漏)。
- **显隐全部**:Shift+F3(toggle_pin)隐藏/显示所有贴图浮窗。
- **复制/保存复用 M2 的 ScreenshotWriter**(copy_to_clipboard / save_to_file,走 config.screenshot)。

**OUT(后续里程碑,本轮占位不实现):**
- **文字识别(OCR)**菜单项 → M4(点了打日志/占位,不报错)。
- OCR 结果窗、翻译 → M4。
- 贴图内容的标注再编辑(贴图上再画)→ 暂不做。
- PyInstaller 打包 → M5。

## 约束
- 框架:**PySide6 (Qt6)**;复用 M1/M2 既有设施:`app_context` 协调器、`config.settings`、`theme/tokens`、
  `components/shadow_widget`、`core/screenshot/writer`(复制/保存)、`core/logger`。
- 目标平台:**Windows 10**;开发机 **Linux + Python 3.14 + headless + 无 pip** → 本机仅 `py_compile` + 代码审查,
  真实贴图/置顶/拖拽/缩放/剪贴板在 **Windows 实测**。
- **全中文 UI**(菜单文案 + 注释中文)。
- 视觉遵循 design token(主色 #2D7FF9、圆角、柔阴影);浮窗/徽标/菜单须还原 mockup 界面②。
- 完全离线;架构与 M1/M2 一致(信号驱动、UI 可扩展);对无剪贴板图 / 无屏幕做防御性降级(不崩,打日志)。
- 与 M2 集成:`screenshot_window` 已发 `sig_result(QImage,"pin")`,app_context 路由到 PinManager。

## 验收标准
1. M2 截图后点「钉图」→ 桌面出现贴图浮窗(置顶、带阴影细边框、右下角缩放徽标)。
2. F3 / 托盘「贴图」「从剪贴板贴图」→ 把剪贴板图片钉到桌面;剪贴板无图时给出提示且不崩。
3. 拖动浮窗可移动;滚轮缩放且徽标实时更新百分比。
4. 右键菜单各项可用:复制(进剪贴板可粘贴)、保存(按 config 写文件)、缩放、不透明度、旋转 90°、置顶切换。
5. 取消贴图 / Esc、销毁 / Shift+Esc 关闭浮窗;关闭后无残留进程/对象泄漏。
6. Shift+F3 隐藏/显示所有贴图;多个贴图可同时存在并独立操作。
7. OCR 菜单项存在且可点击(M4 占位打日志),不报错。
8. 代码结构清晰、命名语义化;关键交互与异常有日志;复制/保存复用 writer,不重复造轮子。

## 跨平台现实
贴图置顶、无边框拖拽、滚轮缩放、剪贴板读图的真实行为强依赖 Windows 桌面。
本机 headless 无法显示窗口 → M3 在 Linux 侧验证「能 py_compile + 纯逻辑(缩放百分比/钳制、菜单装配)+ 代码审查」;
真实贴图/置顶/拖拽/缩放/剪贴板留待 **Windows 10 实测**(README 附 M3 逐条清单)。
